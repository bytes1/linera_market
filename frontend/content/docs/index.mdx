Welcome to True Market, a decentralized prediction market platform powered by MOCA blockchain.


## Introduction

True Market is a modern, open, and intelligent prediction market platform built on the MOCA blockchain. It provides a comprehensive suite of tools for creating markets, trading prediction shares, and engaging with decentralized forecasting.

**Permissionless & Transparent**

True Market uses blockchain to ensure permissionless market creation and transparent price discovery. Anyone can participate, trade, and resolve markets without intermediaries.

### Real-Time Trading

## Our Tech Stack

True Market is built with a modern tech stack to ensure security, scalability, and seamless user experience:

**Frontend:** Next.js, React, TypeScript, Tailwind CSS, shadcn/ui

**Blockchain:** MOCA Chain, EVM-Compatible Smart Contracts, Web3.js / Ethers.js

**Wallet Integration:** Web3 Compatible Wallets

**Backend & Infrastructure:** Vercel, MongoDB, API Routes

**AI & Analytics:** Google Gemini, Real-time Data Processing
